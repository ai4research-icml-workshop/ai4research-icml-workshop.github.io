# AI as a Tool for Mathematics, Computer Science, and Machine Learning

ICML 2026 Workshop website — [ai4research-icml-workshop.github.io](https://ai4research-icml-workshop.github.io)

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Open http://localhost:4000

## Updating Content

- **Speakers**: Edit `_data/speakers.yml`
- **Organizers**: Edit `_data/organizers.yml`
- **Schedule**: Edit `_data/schedule.yml`
- **Other sections**: Edit the corresponding file in `_includes/`

Push to `main` branch to deploy via GitHub Pages.
